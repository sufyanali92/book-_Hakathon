This repository contains the complete textbook on Physical AI and Humanoid Robotics, designed to guide students and practitioners through the intersection of artificial intelligence and physical systems.

## About This Textbook

This textbook is organized into four progressive parts that take you on a journey from basic principles to advanced autonomy:

- **Part I** covers the foundations of Physical AI and robotic systems
- **Part II** explores the ROS 2 framework that powers modern robotics
- **Part III** delves into simulation and digital twins for robot development
- **Part IV** examines vision-language-action models and autonomous decision-making

## Learning Approach

This book follows a simulation-first, hardware-agnostic approach that emphasizes safety and practical understanding. Each chapter builds upon previous concepts while maintaining self-contained explanations for accessibility.

## Target Audience

This textbook is designed for students, beginners to intermediate practitioners, and AI professionals looking to understand how artificial intelligence operates in the physical world through humanoid robots and other robotic systems.

## How to Use This Book

- Follow the chapters sequentially for progressive learning
- Each chapter includes learning objectives, practical examples, and exercises
- Code examples are provided where pedagogically useful
- Concepts are explained with system integration in mind

## Building the Textbook

This textbook is built using Docusaurus. To run it locally:

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run start
```

3. Open your browser to http://localhost:3000
